→ UGUI

오브젝트 옆 문맥 버튼(밀기/부수기/열기/줍기) + 조준·높이 마커. 월드 좌표를 추종해야 하므로 UGUI(Screen Space-Camera). 시안은 배치·톤 참고용.

구현 방침. 위 배지는 docs/UI_ARCHITECTURE.md의 한 줄 기준을 따른다 — 월드 좌표를 따라다니거나 엔티티/타일에 앵커되면 UGUI, 화면공간 평면 패널이면 UI Toolkit. UI Toolkit 화면(01·02·03)은 이 HTML 시안을 UXML(구조)/USS(스타일)로 거의 1:1 이식한다. UGUI 화면(04)은 톤·레이아웃만 참고하고 씬에서 Canvas·RectTransform으로 구성. 실제 Unity 코드는 M4(상호작용 버튼 UI)·M5(UI 폴리시)에서 착수.
